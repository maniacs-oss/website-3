Speak Changelog

  1. v0.3.2

    • New: Dock now grows from center of screen like app dock
    • New: Twitter link in app menu, we really need some followers ya'll!
    • Fixed: Issue where dock would float in the screen instead of being against edge
    • Fixed: Issue where audio continues to be uploaded after call is ended
    • Fixed: Issue where you would stay idle after resuming laptop from sleep
    • Fixed: Calling sound continued after ignoring a call
    • Fixed: Music now resumes when a call finishes as expected
    • Fixed: Improved resource consumption when idle
    • Fixed: CMD+SHIFT+H miswritten in shortcuts preferences
    • Fixed: Updated language around call and accept mode
    • Fixed: Unclickable area on screen when signed out of Speak
    • Fixed: If muted when another user joins a call the UI appears to unmute
    • Fixed: A number of minor display issues and refinements #littlebigdetails

  2. v0.3.1

    • New: Now downloads via dmg installer on Mac
    • New: Added CMD+SHIFT+H global shortcut to leave a call
    • Fixed: Unable to signup with Slack when profile doesn't include an avatar
    • Fixed: No longer sending speaking events whilst calling and unanswered
    • Fixed: Stops ability to drag images into Speak windows and have them load
    • Fixed: Much improved error reporting so we can track bugs down quickly!

  3. v0.3.0

    • New: Change dock to right hand side of screen!
    • New: Option to accept an invite and change team
    • New: Added direct link to email us under 'contact support'
    • Fixed: Undefined user after call
    • Fixed: Now better handling of errors during Slack authentication
    • Fixed: Numerous other small issues and bug fixes

  4. v0.2.9

    • New: Link to this change log!
    • New: Call when idle preference is now enabled by default
    • Fixed: Instances where there would be no audio notification
    • Fixed: Possible to appear muted but not be after leaving a call in mute state
    • Fixed: Ability to clear missed call notifications in a call
    • Fixed: Invite tooltip animation #littlebigdetails

  5. v0.2.8

    • New: Upload your own avatar from the profile preferences
    • New: Option to disable instant audio when idle/away from the computer
    • New: Missing a call will now add a badge to the user so you know to call back
    • New: Clearer denotation when calling a user and not yet connected
    • New: Alt+click on a users picture to audio call them and skip the menu
    • Fixed: Several issues when loading after first signup
    • Fixed: CMD+SHIFT+N shortcut now working
    • Fixed: Improved animations for calls
    • Fixed: General call stability improvements in multiparty
    • Fixed: Slack authentication now has a bigger window size
    • Fixed: Signin link on forgot password screen
